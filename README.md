## 🚀 How to run locally

1. Clone the repository:

```bash
git clone <repo-url>
cd my-portfolio
```

2. Install dependencies:

```bash
npm install
```

3. Run dev server (network accessible):

```bash
npm run dev -- --host
```

4. Open browser at `http://localhost:5173` or local IP to view on mobile.

---

## ⚙️ Environment Variables

Create a `.env` file in root with your settings, for example:

```
VITE_EMAILJS_SERVICE_ID=...
VITE_EMAILJS_TEMPLATE_ID=...
VITE_EMAILJS_USER_ID=...
```
